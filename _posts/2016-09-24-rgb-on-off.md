## Example
This example shows how to switch on and off the RGB LED.

```c++
#include <NanoPlayBoard.h>

NanoPlayBoard board;

void setup() {

}

void loop() {
  board.rgb.on();
  delay(1000);
  board.rgb.off();
  delay(1000);
}
```

The source code of this example is [available on GiHub][1].
